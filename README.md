# Menu Items Display Test
## Installation

- Install related packages

```bash
composer install
```
```bash
npm install
```

- Duplicate file '.env.example' and rename to '.env'
- Update related database config
- Migrate database
```bash
php artisan migrate
```
- Serve the site
```bash
php artisan serve
```
- Run node script
```bash
npm run watch
# or
npm run dev
```
